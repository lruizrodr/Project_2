About This Project

About

This page documents the project goal, scope, and the criteria used for evaluation.

User Stories

  • As a user, I can search for GIFs by keyword.
  • As a user, I can load more results on demand.
  • As a user, I can view a responsive grid of results on mobile and desktop.

Wireframes

High-level layout and navigation structure for Home, About, and Contact views.

Requirements

  • Accept a keyword and query the Giphy API.
  • Parse the response and render images.
  • Provide “Load more” pagination.
  • Host on GitHub Pages with a public repo and README.

Technologies

  • HTML, SCSS/CSS
  • Vanilla JavaScript (fetch, async/await)
  • Giphy API
  • GitHub Pages

Grading Criteria

  • Functionality and robustness
  • Creativity, styling, UX
  • Code quality and structure
  • Documentation and commit history

Setup

  1. Clone repo and open index.html.
  2. Add your Giphy API key to the script file.
  3. Run locally with a simple HTTP server or open directly.

Usage

  1. Enter a term in the search box.
  2. Press Lizardfy to fetch GIFs.
  3. Click Load More to paginate results.

Ideas for Improvement

  • Favorites with local storage
  • Filters (rating, type)
  • Light/Dark/Neon themes
